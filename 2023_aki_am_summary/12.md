# 令和5年度秋期 問12（コンピュータシステム）

## 問題文

SAN（Storage Area Network）におけるサーバとストレージの接続形態の説明として，適切なものはどれか。

ア　シリアルATAなどの接続方式によって内蔵ストレージとして1対1に接続する。

イ　ファイバチャネルなどによる専用ネットワークで接続する。

ウ　プロトコルはCIFS（Common Internet File System）を使用し，LANで接続する。

エ　プロトコルはNFS（Network File System）を使用し，LANで接続する。

## 使用画像

なし

## 解答と解説

**正解：イ**

SAN（Storage Area Network）は、サーバとストレージ装置（ディスクアレイ等）の間を、ファイバチャネル（FC）やiSCSIなどのブロックI/Oプロトコルを使用する専用のネットワークで接続する方式である。複数のサーバから複数のストレージ装置へ柔軟に接続でき、LANとは独立した高速・低遅延な専用ネットワーク上でストレージを共有できる点が特徴であり、これを述べたイが正解である。

他の選択肢について、アはDAS（Direct Attached Storage）の説明であり、サーバに1対1で内蔵・直結する構成でありネットワークを介さない。ウのCIFS（現SMB）はファイル共有プロトコルであり、ウ・エで述べられているようにLAN上でファイル単位のアクセスを行うのはNAS（Network Attached Storage）の特徴である。エのNFSも同様にファイル単位でLAN経由のアクセスを提供するNASのプロトコルであり、SANのようなブロック単位・専用ネットワークでの接続とは異なる。

したがって、ファイバチャネルなどによる専用ネットワークで接続するというイが正解である。

**IPA公式：イ**
