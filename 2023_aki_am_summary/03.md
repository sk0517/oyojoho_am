# 令和5年度秋期 問3（基礎理論）

## 問題文

逆ポーランド表記法（後置記法）で表現されている式ABCD−×＋において，A＝16，B＝8，C＝4，D＝2のときの演算結果はどれか。逆ポーランド表記法による式AB＋は，中置記法による式A＋Bと同一である。

ア　32

イ　46

ウ　48

エ　94

## 使用画像

なし

## 解答と解説

**正解：ア**

逆ポーランド表記法（後置記法）は、演算子をオペランドの後に置く記法であり、スタックを用いて左から順に読み進めることで中置記法へ変換・計算できる。

式 ABCD−×＋ をスタックを使って評価する。

1. A（16）をスタックに積む → [16]
2. B（8）をスタックに積む → [16, 8]
3. C（4）をスタックに積む → [16, 8, 4]
4. D（2）をスタックに積む → [16, 8, 4, 2]
5. 「−」：スタックから2つ取り出し C−D = 4−2 = 2 を計算し積む → [16, 8, 2]
6. 「×」：スタックから2つ取り出し B×(C−D) = 8×2 = 16 を計算し積む → [16, 16]
7. 「＋」：スタックから2つ取り出し A＋(B×(C−D)) = 16＋16 = 32 を計算する

したがって中置記法では A ＋ (B × (C − D)) となり、演算結果は 32 である。これは選択肢アに一致する。

**IPA公式：ア**
