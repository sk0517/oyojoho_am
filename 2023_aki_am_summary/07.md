# 令和5年度秋期 問7（基礎理論）

## 問題文

JavaScriptのオブジェクトの表記法などを基にして規定したものであって，“名前と値との組みの集まり”と“値の順序付きリスト”の二つの構造に基づいてオブジェクトを表現する，データ記述の仕様はどれか。

ア　DOM

イ　JSON

ウ　SOAP

エ　XML

## 使用画像

なし

## 解答と解説

**正解：イ**

JSON（JavaScript Object Notation）は、JavaScriptのオブジェクトリテラル表記法を基にして規定された軽量なデータ記述形式である。JSONは「名前と値の組みの集まり（オブジェクト、`{ }`で表現）」と「値の順序付きリスト（配列、`[ ]`で表現）」という二つの構造を基本としてデータを表現する。人間にも読みやすく、プログラムでも解析しやすいことから、Web APIなどでのデータ交換形式として広く使われている。

- ア DOM（Document Object Model）：HTMLやXML文書をツリー構造のオブジェクトとして扱うためのAPI仕様であり、データ記述形式そのものではない。
- ウ SOAP：XMLベースのメッセージ交換プロトコルであり、名前と値の組み・順序付きリストという構造を規定するものではない。
- エ XML：タグによる階層構造でデータを記述する言語であり、JavaScriptのオブジェクト表記法を基にしたものではない。

以上より、正解はイ（JSON）である。

**IPA公式：イ**
