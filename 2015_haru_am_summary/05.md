# 平成27年度春期 問5（基礎理論）

## 問題文

自然数をキーとするデータを，ハッシュ表を用いて管理する。キーxのハッシュ関数h(x)を

　　　　h(x)＝x mod n

とすると，キーaとbが衝突する条件はどれか。ここで，nはハッシュ表の大きさであり，x mod nはxをnで割った余りを表す。

ア　a＋bがnの倍数

イ　a−bがnの倍数

ウ　nがa＋bの倍数

エ　nがa−bの倍数

## 使用画像

なし

## 解答と解説

**正解：イ**

ハッシュ関数h(x) = x mod n は、キーxをnで割った余りをハッシュ値とする方式である。キーaとbが衝突するとは、h(a) = h(b)、すなわちaをnで割った余りとbをnで割った余りが等しいことを意味する。

a mod n = b mod n が成り立つとき、a とb はnを法として合同（a ≡ b (mod n)）であり、これは (a − b) がnの倍数であることと同値である。

具体例で確認すると、n=5のときa=7, b=12とすると、7 mod 5 = 2、12 mod 5 = 2で衝突する。このときa−b = −5 = −1×5 であり、確かにnの倍数になっている。

- ア：a＋bがnの倍数という条件は、衝突条件（余りの一致）とは異なる。例えばa=2,b=3,n=5だとa+b=5でnの倍数だが、2 mod 5=2、3 mod 5=3で衝突しない。
- イ：a−bがnの倍数、という条件が衝突の必要十分条件であり正しい。
- ウ、エ：「nがa+bの倍数」「nがa−bの倍数」は主客が逆で、一般には成立しない。

以上より、正解はイである。

**IPA公式：イ**
