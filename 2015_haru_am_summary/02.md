# 平成27年度春期 問2（基礎理論）

## 問題文

2桁の2進数x_1x_2が表す整数をxとする。2進数x_2x_1が表す整数を，xの式で表したものはどれか。ここで，int(r)は非負の実数rの小数点以下を切り捨てた整数を表す。

ア　2x＋4int (x／2)

イ　2x＋5int (x／2)

ウ　2x−3int (x／2)

エ　2x−4int (x／2)

## 使用画像

なし

## 解答と解説

**正解：ウ**

2桁の2進数x1x2（x1が上位ビット、x2が下位ビット）が表す整数をxとすると、
　x = 2×x1 + x2

x1, x2はそれぞれ0または1なので、int(x/2)を考えると、
- x1=0, x2=0のとき x=0, int(x/2)=0
- x1=0, x2=1のとき x=1, int(x/2)=0
- x1=1, x2=0のとき x=2, int(x/2)=1
- x1=1, x2=1のとき x=3, int(x/2)=1

つまり int(x/2) = x1 であることが分かる。

桁を入れ替えた2進数x2x1が表す整数yは、
　y = 2×x2 + x1

ここでx1 = int(x/2)なので、
　y = 2×x2 + int(x/2)

さらに x2 = x − 2×x1 = x − 2×int(x/2) であるから、これを代入すると、
　y = 2×(x − 2×int(x/2)) + int(x/2)
　　= 2x − 4×int(x/2) + int(x/2)
　　= 2x − 3×int(x/2)

これは選択肢ウの「2x−3int(x/2)」と一致する。実際に4パターンで検算すると、
- x1x2=00 (x=0) → x2x1=00 (y=0)：2×0−3×int(0/2)=0 ✓
- x1x2=01 (x=1) → x2x1=10 (y=2)：2×1−3×int(1/2)=2−0=2 ✓
- x1x2=10 (x=2) → x2x1=01 (y=1)：2×2−3×int(2/2)=4−3=1 ✓
- x1x2=11 (x=3) → x2x1=11 (y=3)：2×3−3×int(3/2)=6−3=3 ✓

全て一致するため、正解はウである。

**IPA公式：ウ**
