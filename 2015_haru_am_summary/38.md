# 平成27年度春期 問38（技術要素）

## 問題文

パスワードに使用できる文字の種類の数をM，パスワードの文字数をnとするとき，設定できるパスワードの理論的な総数を求める数式はどれか。

ア　Mn

イ　M!(M−n)!

ウ　M!n!(M−n)!

エ　(M＋n−1)!n!(M−1)!

    span.p-divition-number-text {
        font-size: 90%;
        line-height: 80%;
    }

    table.p-division-number{

    }

なし

## 解答と解説

**正解：ア**

パスワードは、使用できる文字の種類の数M種類の中から、n文字分を選んで並べる組合せである。各桁（各文字位置）について独立にM種類の文字のいずれかを選べるため、理論的に設定可能なパスワードの総数は、Mをn回掛け合わせた値、すなわちMのn乗（M^n）で求められる。

例えば、数字10種類（M＝10）を使って4桁（n＝4）のパスワードを作る場合、10^4＝10,000通りとなり、これは0000～9999までの組合せ数と一致し、直感的にも妥当である。

他の選択肢にある階乗（！）を用いた式は、同じ文字を重複して使わずに全て異なる文字を並べる順列（重複なし順列）を求める場合などに使われる考え方であり、パスワードのように同じ文字種を繰り返し使用できる設定においては適切でない。したがって、文字の重複使用を許すパスワードの理論的総数を求める式としては、単純な累乗であるMのn乗（選択肢ア）が正しい。

**IPA公式：ア**
