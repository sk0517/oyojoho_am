# 平成27年度春期 問3（基礎理論）

## 問題文

製品100個を1ロットとして生産する。一つのロットからサンプルを3個抽出して検査し，3個とも良品であればロット全体を合格とする。100個中に10個の不良品を含むロットが合格と判定される確率は幾らか。

ア　710

イ　178245

ウ　7291000

エ　89110

## 解答と解説

**正解：イ**

100個中10個が不良品（残り90個が良品）のロットから、非復元抽出で3個をサンプル抽出し、3個とも良品である確率を求める。

3個とも良品になる確率は、超幾何分布の考え方で、
　P = (90/100) × (89/99) × (88/98)

これを計算すると、
　P = (90×89×88) / (100×99×98)
　　= 704,880 / 970,200

分子・分母を約分すると、
　= 8×89×88 / (100×99×98/90)…と直接約分するより、組合せで考えた方が明快である。

組合せで表すと、
　P = C(90,3) / C(100,3)

C(90,3) = 90×89×88/6 = 117,480
C(100,3) = 100×99×98/6 = 161,700

　P = 117,480 / 161,700

分子・分母を60で割ると、
　= 1,958 / 2,695

さらに約分を進めると分母は2,695 = 5×7×7×11、分子1,958 = 2×11×89 であるため、11で約分できる。
　= 178 / 245

これは選択肢イの「178/245」と一致する。

- ア：710（分数の書式が崩れているが7/10相当）は明らかに桁が合わない。
- ウ：729/1000も本問の組合せ計算とは一致しない。
- エ：89/110も計算結果と異なる。

以上より、正解はイである。

**IPA公式：イ**
