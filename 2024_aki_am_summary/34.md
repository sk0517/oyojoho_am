# 令和6年度秋期 問34（技術要素）

## 問題文

IPv4ネットワークにおけるICMPの機能として，適切なものはどれか。

ア　MACアドレスだけが分かっているときにIPアドレスの解決を可能にする。

イ　グローバルIPアドレスとプライベートIPアドレスとを相互に変換する。

ウ　送信元ホストへ，IPパケットの送信エラー報告などの制御メッセージを通知する。

エ　ネットワーク内のIPアドレスを一元管理し，クライアントに動的に割り当てる。

## 使用画像

なし

## 解答と解説

**正解：ウ**

ICMP（Internet Control Message Protocol）は，IPパケットの転送における誤りやネットワークの状態を通知するためのプロトコルである。IPパケットが宛先に到達できなかった場合などに，送信元ホストへエラー報告や制御メッセージ（宛先到達不能，時間超過など）を通知する機能をもつ。pingコマンドで使われるecho要求／応答もICMPメッセージの一種である。したがって，ウが正しい。

アはARP（Address Resolution Protocol）の機能である（実際はIPアドレスからMACアドレスを解決するのが一般的なARPの用途）。イはNAT（Network Address Translation）の機能である。エはDHCP（Dynamic Host Configuration Protocol）の機能である。いずれもICMPとは異なるプロトコルの機能である。

**IPA公式：ウ**
