# 令和6年度秋期 問14（コンピュータシステム）

## 問題文

稼働率が等しい装置Xを直列や並列に組み合わせたとき，システム全体の稼働率を高い順に並べたものはどれか。ここで，装置Xの稼働率は0よりも大きく1未満である。

ア　A，B，C

イ　A，C，B

ウ　C，A，B

エ　C，B，A

## 使用画像

![AP2024AA014-01.gif](images/AP2024AA014-01.gif)

## 解答と解説

**正解：イ**

装置Xの稼働率をrとする（0＜r＜1）。画像の3つの構成の稼働率を求める。

- 構成A：装置Xを2台並列にした構成。稼働率＝1－(1－r)²
- 構成B：装置Xを1台と、装置Xを2台並列にしたブロックを直列に接続した構成。稼働率＝r×｛1－(1－r)²｝
- 構成C：装置Xを2台並列にしたブロックを2組、直列に接続した構成。稼働率＝｛1－(1－r)²｝²

ここでA＝1－(1－r)²とおくと、B＝r×A、C＝A×Aと表せる。0＜r＜1のとき、A＝2r－r²＝r(2－r)であり、0＜r＜1では(2－r)＞1なのでA＞rが成り立つ。

- A と C の比較：C＝A×A、A＝A×1。A＜1なのでC＝A²＜A、つまりA＞C。
- B と C の比較：C＝A×A、B＝A×r。A＞rなのでC＝A×A＞A×r＝B、つまりC＞B。

以上より稼働率の大小関係はA＞C＞Bとなる。例えばr＝0.5のとき、A＝0.75、C＝0.5625、B＝0.375となり、この関係が確認できる。したがって高い順に並べるとA，C，Bとなり、イが正解である。

**IPA公式：イ**
