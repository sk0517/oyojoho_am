# 平成29年度秋期 問38（技術要素）

## 問題文

SIEM（Security Information and Event Management）の特徴はどれか。

ア　DMZを通過する全ての通信データを監視し，不正な通信を遮断する。

イ　サーバやネットワーク機器のMIB（Management Information Base）情報を分析し，中間者攻撃を遮断する。

ウ　ネットワーク機器のIPFIX（IP Flow Information Export）情報を監視し，攻撃者が他者のPCを不正に利用したときの通信を検知する。

エ　複数のサーバやネットワーク機器のログを収集分析し，不審なアクセスを検知する。

## 使用画像

なし

## 解答と解説

**正解：エ**

SIEM（Security Information and Event Management）は、ファイアウォール、IDS/IPS、サーバ、ネットワーク機器など複数の機器から出力されるログ情報を一元的に収集・相関分析し、単体のログだけでは気付きにくい不審なアクセスや攻撃の兆候を検知する仕組みである。

- ア：これはファイアウォール（特にDMZ境界での通信制御）の説明であり、SIEMのようなログの収集・分析とは異なる。
- イ：MIB情報の分析はSNMPによる機器監視に関するものであり、中間者攻撃の遮断とは直接結び付かない。
- ウ：IPFIX情報の監視はネットワークフロー分析の手法の一つであり、SIEMが行う複数ログの統合分析とは範囲が異なる。

以上より、正解はエである。

**IPA公式：エ**
