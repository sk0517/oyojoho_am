# 平成29年度秋期 問52（マネジメント）

## 問題文

プロジェクト管理においてパフォーマンス測定に使用するEVMの管理対象の組みはどれか。

ア　コスト，スケジュール

イ　コスト，リスク

ウ　スケジュール，品質

エ　品質，リスク

## 使用画像

なし

## 解答と解説

**正解：ア**

EVM（Earned Value Management：出来高管理）は、プロジェクトの進捗とコストを金額（出来高）という共通の尺度で定量的に評価する管理手法である。代表的な指標にPV（計画価値）、EV（出来高）、AC（実コスト）があり、これらからSV（スケジュール差異）やCV（コスト差異）を算出することで、スケジュールとコストの両面のパフォーマンスを同時に把握できる。したがって、EVMの管理対象は「コスト，スケジュール」の組み合わせであり、選択肢アが正解となる。

品質やリスクはEVMの直接の測定対象ではなく、それぞれ品質管理・リスク管理という別の管理プロセスで扱われるため、イ・ウ・エは誤り。

**IPA公式：ア**
