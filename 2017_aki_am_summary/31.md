# 平成29年度秋期 問31（技術要素）

## 問題文

無線LANで用いられるSSIDの説明として，適切なものはどれか。

ア　48ビットのネットワーク識別子であり，アクセスポイントのMACアドレスと一致する。

イ　48ビットのホスト識別子であり，有線LANのMACアドレスと同様の働きをする。

ウ　最長32オクテットのネットワーク識別子であり，接続するアクセスポイントの選択に用いられる。

エ　最長32オクテットのホスト識別子であり，ネットワーク上で一意である。

## 使用画像

なし

## 解答と解説

**正解：ウ**

SSID（Service Set Identifier）は、無線LANにおいてアクセスポイント（またはネットワーク）を識別するための最長32オクテット（32バイト）の識別子であり、端末が接続先のアクセスポイントを選択する際に使用される。同一のSSIDを設定した複数のアクセスポイントでローミングを構成することもできる。

- ア：誤り。48ビットの識別子はMACアドレスであり、SSIDとは異なる仕組みである。
- イ：誤り。SSIDはホスト識別子ではなくネットワーク（アクセスポイント）識別子であり、ビット長も48ビットではない。
- エ：誤り。SSIDは「ネットワーク識別子」であって「ホスト識別子」ではない。また同一ネットワーク内で複数の機器が同じSSIDを使うため、一意性は保証されない。

以上より、正解はウである。

**IPA公式：ウ**
