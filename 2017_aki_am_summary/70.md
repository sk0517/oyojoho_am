# 平成29年度秋期 問70（ストラテジ）

## 問題文

構成表の製品Aを300個出荷しようとするとき，部品bの正味所要量は何個か。ここで，A，a，b，cの在庫量は在庫表のとおりとする。また，他の仕掛残，注文残，引当残などはないものとする。

　

ア　200

イ　600

ウ　900

エ　1,500

## 解答と解説

**正解：イ**

構成表より、製品Aは部品aを3個、部品bを2個必要とする。さらに部品aは部品bを1個必要とする（部品cはA・aとも所要だが今回は問われていない）。在庫表より、A=100個、a=100個、b=300個、c=400個である。

1. 製品Aの正味所要量：出荷300個 − 在庫100個 = 200個
2. 部品aの総所要量：Aの正味所要量200個 × 3（Aあたりの必要数）= 600個
   部品aの正味所要量：600個 − 在庫100個 = 500個
3. 部品bの総所要量：
   - Aから直接必要な分：Aの正味所要量200個 × 2 = 400個
   - aから必要な分：aの正味所要量500個 × 1 = 500個
   - 合計：400個 + 500個 = 900個
4. 部品bの正味所要量：総所要量900個 − 在庫300個 = 600個

したがって、部品bの正味所要量は600個であり、正解はイである。

**IPA公式：イ**
