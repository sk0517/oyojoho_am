# 平成29年度秋期 問49（開発技術）

## 問題文

CMMIの説明はどれか。

ア　ソフトウェア開発組織及びプロジェクトのプロセスの成熟度を評価するためのモデルである。

イ　ソフトウェア開発のプロセスモデルの一種である。

ウ　ソフトウェアを中心としたシステム開発及び取引のための共通フレームのことである。

エ　プロジェクトの成熟度に応じてソフトウェア開発の手順を定義したモデルである。

## 使用画像

なし

## 解答と解説

**正解：ア**

CMMI（Capability Maturity Model Integration：能力成熟度モデル統合）は、組織のソフトウェア開発やシステム開発のプロセスがどの程度成熟しているかを5段階のレベルで評価し、プロセス改善の指針とするためのモデルである。

- イ：ウォーターフォールモデルやアジャイルなど「開発プロセスモデル」の説明であり、CMMIとは異なる。
- ウ：共通フレーム（SLCP-JCF）の説明である。
- エ：CMMIはプロセスの成熟度を評価するものであり、成熟度に応じて開発手順を定義するモデルではない。

**IPA公式：ア**
