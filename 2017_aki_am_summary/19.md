# 平成29年度秋期 問19（コンピュータシステム）

## 問題文

Hadoopの説明はどれか。

ア　Java EE仕様に準拠したアプリケーションサーバ

イ　LinuxやWindowsなどの様々なプラットフォーム上で動作するWebサーバ

ウ　機能の豊富さが特徴のRDBMS

エ　大規模なデータを分散処理するためのソフトウェアライブラリ

## 使用画像

なし

## 解答と解説

**正解：エ**

Hadoopは、大規模なデータセットを複数のサーバ（クラスタ）に分散させ、並列処理によって効率よく処理するためのオープンソースのソフトウェアライブラリ（フレームワーク）である。分散ファイルシステムHDFS（Hadoop Distributed File System）と、分散処理を行うMapReduceを中核として構成され、ビッグデータ処理の基盤技術として広く利用されている。

- ア：Java EE仕様に準拠したアプリケーションサーバは、例えばApache TomcatやWildFlyなどが該当し、Hadoopとは目的も仕組みも異なる。
- イ：様々なプラットフォームで動作するWebサーバは、Apache HTTP ServerやNginxなどの説明であり、Hadoopとは無関係。
- ウ：機能の豊富なRDBMSは、Oracle DatabaseやPostgreSQLなどのリレーショナルデータベース管理システムの説明であり、Hadoopは分散処理フレームワークであってRDBMSではない。

**IPA公式：エ**
