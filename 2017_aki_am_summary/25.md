# 平成29年度秋期 問25（技術要素）

## 問題文

W3Cで仕様が定義され，矩（く）形や円，直線，文字列などの図形オブジェクトをXML形式で記述し，Webページでの図形描画にも使うことができる画像フォーマットはどれか。

ア　OpenGL

イ　PNG

ウ　SVG

エ　TIFF

## 使用画像

なし

## 解答と解説

**正解：ウ**

SVG（Scalable Vector Graphics）は、W3Cによって仕様が策定されたベクター画像フォーマットであり、矩形、円、直線、テキストなどの図形オブジェクトをXML形式のテキストで記述する。ベクター形式のため拡大・縮小しても画質が劣化せず、Webページ上でスクリプトやスタイルシートと組み合わせて動的な図形描画にも利用できる。

- ア：OpenGLは3Dグラフィックス描画のためのAPI（プログラミングインタフェース）であり、画像フォーマットではない。
- イ：PNG（Portable Network Graphics）はビットマップ（ラスタ）形式の画像フォーマットであり、XMLベースのベクター記述ではない。
- エ：TIFF（Tagged Image File Format）もラスタ形式の画像フォーマットであり、主に印刷・スキャン画像などで使われる。

以上より、W3C仕様・XML形式・ベクター図形の条件をすべて満たすウ（SVG）が正解である。

**IPA公式：ウ**
