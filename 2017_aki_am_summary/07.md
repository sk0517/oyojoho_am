# 平成29年度秋期 問7（基礎理論）

## 問題文

fact（n）は，非負の整数nに対してnの階乗を返す。fact（n）の再帰的な定義はどれか。

ア　if　n＝0　then　return　0　else　return　n×fact（n−1）

イ　if　n＝0　then　return　0　else　return　n×fact（n＋1）

ウ　if　n＝0　then　return　1　else　return　n×fact（n−1）

エ　if　n＝0　then　return　1　else　return　n×fact（n＋1）

## 使用画像

なし

## 解答と解説

**正解：ウ**

階乗の定義は0!＝1、n!＝n×(n−1)!（n≥1）である。再帰関数として書くと、基底ケース（n＝0）でreturn 1、それ以外はn×fact(n−1)を返す必要がある。

選択肢ウ「if n＝0 then return 1 else return n×fact(n−1)」がこの定義に正確に一致する。例えばfact(3)＝3×fact(2)＝3×2×fact(1)＝3×2×1×fact(0)＝3×2×1×1＝6となり、正しく3!が計算できる。

- ア：n＝0のときreturn 0としており、fact(0)＝0となってしまう。これでは全ての階乗が0になり誤り（0が定義される基底ケースが間違っている）。
- イ：n＝0のときreturn 0かつfact(n＋1)を呼んでおり、nが増加し続けて再帰が停止しない。
- エ：基底ケースはreturn 1で正しいが、fact(n＋1)を呼ぶため再帰がn＝0に収束せず無限再帰になる。

**IPA公式：ウ**
