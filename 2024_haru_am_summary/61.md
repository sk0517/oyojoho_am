# 令和6年度春期 問61（ストラテジ）

## 問題文

エンタープライズアーキテクチャの参照モデルのうち，BRM（Business Reference Model）として提供されるものはどれか。

ア　アプリケーションサービスを機能的な観点から分類・体系化したサービスコンポーネント

イ　サービスコンポーネントを実際に活用するためのプラットフォームやテクノロジの標準仕様

ウ　参照モデルの中で最も業務に近い階層として提供される，業務分類に従った業務体系及びシステム体系と各種業務モデル

エ　組織間で共有される可能性の高い情報について，名称，定義及び各種属性を総体的に記述したモデル

## 使用画像

なし

## 解答と解説

**正解：ウ**

BRM（Business Reference Model）は、政府や企業のエンタープライズアーキテクチャ（EA）参照モデルの一つで、参照モデルの階層の中で最も業務（ビジネス）に近い位置付けとなる。業務分類に従って整理された業務体系（機能・プロセス）及びシステム体系、各種業務モデルを提供するものであり、選択肢ウが該当する。

なお、アはSRM（Service Component Reference Model）、イはTRM（Technical Reference Model）、エはDRM（Data Reference Model）の説明に相当する。EAの参照モデルにはこのほかPRM（業績参照モデル）があり、これらを組み合わせて業務とITの全体最適化を図る。

**IPA公式：ウ**
