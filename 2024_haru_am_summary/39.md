# 令和6年度春期 問39（技術要素）

## 問題文

自社製品の脆（ぜい）弱性に起因するリスクに対応するための社内機能として，最も適切なものはどれか。

ア　CSIRT

イ　PSIRT

ウ　SOC

エ　WHOISデータベースの技術連絡担当

## 使用画像

なし

## 解答と解説

**正解：イ**

PSIRT（Product Security Incident Response Team）は，自社が開発・提供する製品（ハードウェア・ソフトウェア）における脆弱性の情報収集，分析，対応，脆弱性情報の公表やパッチ提供などを専門に行う社内組織である。自社製品に起因するリスクへの対応という点で，本問の内容に合致する。

- ア：CSIRT（Computer Security Incident Response Team）は，自組織内の情報システムにおけるセキュリティインシデントに対応する組織であり，主眼は自組織のIT環境であって，自社が「製品として」出荷したものの脆弱性対応に特化した組織ではない。
- ウ：SOC（Security Operation Center）は，ネットワークやシステムを24時間監視し，サイバー攻撃の検知・分析を行う組織であり，自社製品の脆弱性対応が主目的ではない。
- エ：WHOISデータベースの技術連絡担当は，ドメイン名の技術的な問い合わせ窓口であり，製品脆弱性対応の社内機能ではない。

自社製品の脆弱性リスクに特化した対応組織はPSIRTであるため，イが正しい。

**IPA公式：イ**
