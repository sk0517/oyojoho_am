# 平成29年度春期 問3（基礎理論）

## 問題文

ノードとノードの間のエッジの有無を，隣接行列を用いて表す。ある無向グラフの隣接行列が次の場合，グラフで表現したものはどれか。ここで，ノードを隣接行列の行と列に対応させて，ノード間にエッジが存在する場合は1で，エッジが存在しない場合は0で示す。

ア　

イ　

ウ　

エ

## 解答と解説

**正解：ウ**

隣接行列（画像1枚目）からエッジの有無を読み取ると，1が立っているのは次の組合せである。

- a－b
- b－c，b－d
- c－d，c－e
- d－e
- e－f

つまりエッジは {a-b, b-c, b-d, c-d, c-e, d-e, e-f} の7本であり，各ノードの次数は a＝1，b＝3（a, c, d），c＝3（b, d, e），d＝3（b, c, e），e＝3（c, d, f），f＝1 となる。

選択肢の図の中で，a-b-c-d-e-fの直線状の接続に加えて，b-dおよびc-eの2本の弧（曲線）が追加されており，b-cの直線エッジも保持されているものが，上記の隣接関係と一致する。この条件を満たすのがウの図であり，各ノードの次数（a=1, b=3, c=3, d=3, e=3, f=1）も隣接行列と完全に一致する。

他の選択肢は，b-cのエッジが欠けていたり，d-eの直線エッジが欠けているなど，隣接行列が示す辺の構成と一致しない。したがって正解はウである。

**IPA公式：ウ**
