# 平成29年度春期 問50（開発技術）

## 問題文

日本において特許Aを取得した特許権者から，実施許諾を受けることが必要になるのはどれか。

ア　出願日から25年を超えた特許Aと同じ技術を新たに事業化する場合

イ　特許Aの出願日よりも前から特許Aと同じ技術を独自に開発して，特許Aの出願日に日本国内でその技術を用いた製品を製造し，市場で販売していたことが証明できる場合

ウ　特許Aを家庭内で個人的に利用するだけの場合

エ　日本国内で製造し，米国に輸出する製品に特許Aを利用する場合

## 使用画像

なし

## 解答と解説

**正解：エ**

特許権は、原則として権利を取得した国の中でのみ効力を持つ（属地主義）。したがって、日本で取得した特許Aの効力は日本国内にのみ及び、他国では原則としてその国で別途特許を取得しない限り権利は及ばない。しかし、日本国内で「製造」する行為自体は、日本の特許権の効力が及ぶ実施行為（生産）に該当するため、輸出目的であっても国内で特許発明を利用した製品を製造するには、特許権者からの実施許諾が必要になる。エはこの内容に該当する。

アは特許権の存続期間（出願から原則20年）を超えているため、特許権はすでに消滅しており、誰でも自由に実施でき許諾は不要である。イは「先使用権」に該当するケースで、特許出願前から独自に同じ技術を用いた事業を行っていたことを証明できる場合、法定の通常実施権（先使用による通常実施権）が認められ、許諾なく実施を継続できる。ウは特許発明の実施が「業として」行われる場合にのみ特許権の効力が及ぶため、家庭内での個人的・非営業的な利用には特許権の効力が及ばず、許諾は不要である。

以上より、実施許諾を受けることが必要になるのはエである。

**IPA公式：エ**
