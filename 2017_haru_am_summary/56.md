# 平成29年度春期 問56（マネジメント）

## 問題文

新システムの開発を計画している。提案された4案の中で，TCO（総所有費用）が最小のものはどれか。ここで，このシステムは開発後，3年間使用されるものとする。

ア　A案

イ　B案

ウ　C案

エ　D案

## 使用画像

![AP2017SA056-01.gif](images/AP2017SA056-01.gif)

## 解答と解説

**正解：ウ**

TCO（Total Cost of Ownership：総所有費用）は、システムの初期導入費用（イニシャルコスト）と、運用開始後にかかる年間費用（ランニングコスト）の累計を合算した総費用である。本問ではシステムを3年間使用するため、初期費用＋（年間費用×3年）で各案のTCOを求める。

各案の初期費用（ハードウェア導入費用＋システム開発費用＋導入教育費用）は次のとおり。
- A案：30＋30＋5＝65百万円
- B案：30＋50＋5＝85百万円
- C案：40＋30＋5＝75百万円
- D案：40＋40＋5＝85百万円

各案の年間費用（ネットワーク通信費用＋保守費用＋システム運用費用）は次のとおり。
- A案：20＋6＋6＝32百万円／年 → 3年で96百万円
- B案：20＋5＋4＝29百万円／年 → 3年で87百万円
- C案：15＋5＋6＝26百万円／年 → 3年で78百万円
- D案：15＋5＋4＝24百万円／年 → 3年で72百万円

したがって3年間のTCOは次のとおりとなる。
- A案：65＋96＝161百万円
- B案：85＋87＝172百万円
- C案：75＋78＝153百万円
- D案：85＋72＝157百万円

最小のTCOはC案の153百万円であり、正解はウである。

**IPA公式：ウ**
