# 平成29年度春期 問32（技術要素）

## 問題文

ネットワーク機器の接続状態を調べるためのコマンドpingが用いるプロトコルはどれか。

ア　DHCP

イ　ICMP

ウ　SMTP

エ　SNMP

## 使用画像

なし

## 解答と解説

**正解：イ**

pingコマンドは、ネットワーク機器の接続状態や到達性を確認するためのコマンドであり、ICMP（Internet Control Message Protocol）のEcho RequestメッセージとEcho Replyメッセージを利用して疎通確認を行う。送信元がEcho Requestを送り、宛先ホストがEcho Replyを返すことで、ネットワークの応答時間（RTT）やパケットロスの有無を確認できる。したがってイが正解である。

ア「DHCP」は、IPアドレスなどのネットワーク設定情報をクライアントに自動的に割り当てるためのプロトコルであり、pingとは無関係である。
ウ「SMTP」は、電子メールの送信に用いられるプロトコルである。
エ「SNMP」は、ネットワーク機器の状態監視や管理を行うためのプロトコルであり、MIB情報の取得などに用いられるが、pingが直接利用するプロトコルではない。

以上より、pingが用いるプロトコルはイのICMPである。

**IPA公式：イ**
