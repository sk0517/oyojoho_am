# 平成29年度春期 問61（ストラテジ）

## 問題文

情報戦略の投資効果を評価するとき，利益額を分子に，投資額を分母にして算出するものはどれか。

ア　EVA

イ　IRR

ウ　NPV

エ　ROI

## 使用画像

なし

## 解答と解説

**正解：エ**

情報戦略の投資効果を評価する指標のうち、利益額を分子、投資額を分母にして算出するものはROI（Return On Investment：投資利益率）である。ROI＝利益額÷投資額×100（％）で求められ、投じた投資額に対してどれだけの利益を生み出したかを示す代表的な投資効果測定指標である。

- ア　EVA（Economic Value Added：経済的付加価値）は、税引後営業利益から資本コストを差し引いて算出する指標であり、利益額÷投資額の形式ではない。
- イ　IRR（Internal Rate of Return：内部収益率）は、投資案の正味現在価値がちょうどゼロになる割引率を求める指標であり、比率計算ではあるが「利益額÷投資額」の単純な形式ではない。
- ウ　NPV（Net Present Value：正味現在価値）は、将来のキャッシュフローを現在価値に割り引いた合計から投資額を差し引いた金額（絶対額）であり、比率ではない。

したがって、利益額を分子、投資額を分母とする最も単純な投資効果評価指標はエのROIである。

**IPA公式：エ**
