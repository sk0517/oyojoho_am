# 平成29年度春期 問8（コンピュータシステム）

## 問題文

CPUのプログラムレジスタ（プログラムカウンタ）の役割はどれか。

ア　演算を行うために，メモリから読み出したデータを保持する。

イ　条件付き分岐命令を実行するために，演算結果の状態を保持する。

ウ　命令のデコードを行うために，メモリから読み出した命令を保持する。

エ　命令を読み出すために，次の命令が格納されたアドレスを保持する。
