# 平成29年度春期 問17（コンピュータシステム）

## 問題文

サーバアプリケーションの開発のための，オブジェクト指向技術に基づいたコンポーネントソフトウェアの仕様はどれか。

ア　EAI（Enterprise Application Integration）

イ　EJB（Enterprise JavaBeans）

ウ　ERP（Enterprise Resource Planning）

エ　UML（Unified Modeling Language）

## 使用画像

なし

## 解答と解説

**正解：イ**

EJB（Enterprise JavaBeans）は、Javaプラットフォーム上でサーバサイドアプリケーションを構築するための、オブジェクト指向技術に基づいたコンポーネントソフトウェアの仕様である。ビジネスロジックをコンポーネント（Beans）として実装し、トランザクション管理やセキュリティなどのサービスをコンテナが提供することで、再利用性・拡張性の高いサーバアプリケーション開発を可能にする。

ア のEAIは、異なる業務システム間でデータやプロセスを連携・統合するための仕組みであり、コンポーネント仕様ではない。

ウ のERPは、企業の基幹業務（会計、人事、生産、販売など）を統合的に管理する経営資源計画の考え方・システムであり、開発仕様ではない。

エ のUMLは、オブジェクト指向設計・分析に用いるモデリング言語であり、コンポーネントソフトウェアの仕様そのものではない。

したがって、オブジェクト指向に基づくサーバコンポーネントの仕様として正しいのは イ である。

**IPA公式：イ**
