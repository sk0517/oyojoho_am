# 平成29年度春期 問6（基礎理論）

## 問題文

次の流れ図の処理で，終了時のxに格納されているものはどれか。ここで，与えられたa，bは正の整数であり，mod（x，y）はxをyで割った余りを返す。

ア　aとbの最小公倍数

イ　aとbの最大公約数

ウ　aとbの小さい方に最も近い素数

エ　aをbで割った商
