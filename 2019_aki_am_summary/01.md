# 秋期 問1（基礎理論）

## 問題文

あるホテルは客室を1,000部屋もち，部屋番号は，数字4と9を使用しないで0001から順に数字4桁の番号としている。部屋番号が0330の部屋は，何番目の部屋か。

ア　204

イ　210

ウ　216

エ　218

## 使用画像

なし

## 解答と解説

**正解：ウ**

部屋番号には4桁の数字が使われるが，各桁で4と9を除いた8種類の数字（0,1,2,3,5,6,7,8）だけを使用している。これは，使用可能な数字の個数が8個であることから，8進数の桁構成と本質的に同じ仕組みになっている。

そこで，使用されている数字を8進数の各桁に対応付ける（0→0，1→1，2→2，3→3，5→4，6→5，7→6，8→7）。部屋番号「0330」の各桁はそのまま0,3,3,0であり，8進数として読み替えても数字自体は変わらない（4,9が含まれないため）。

「0330」を8進数とみなして10進数に変換すると，
0330(8) = 3×8² + 3×8¹ + 0×8⁰ = 3×64 + 3×8 + 0 = 192 + 24 = 216

部屋番号「0001」が1番目の部屋（8進数の1 = 10進数の1）に対応するため，8進数変換値がそのまま「何番目か」を表す。したがって「0330」は216番目の部屋となる。

他の選択肢（204，210，218）は，桁の重み計算を誤るか，4・9を除外する対応付けを誤った場合に生じやすい値である。

**IPA公式：ウ**
