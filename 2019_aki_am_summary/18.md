# 秋期 問18（コンピュータシステム）

## 問題文

ページング方式の仮想記憶において，主記憶に存在しないページをアクセスした場合の処理や状態の順番として，適切なものはどれか。ここで，現在主記憶には，空きページはないものとする。

ア　置換え対象ページの決定 → ページイン → ページフォールト → ページアウト

イ　置換え対象ページの決定 → ページフォールト → ページアウト → ページイン

ウ　ページフォールト → 置換え対象ページの決定 → ページアウト → ページイン

エ　ページフォールト → 置換え対象ページの決定 → ページイン → ページアウト

## 使用画像

なし

## 解答と解説

**正解：ウ**

ページング方式の仮想記憶において、参照したページが主記憶上に存在しない場合の処理順序を問う問題である。

1. **ページフォールト**：CPUがアクセスしようとしたページが主記憶に存在しないことを検知し、割込み（ページフォールト割込み）が発生する。これが一連の処理の起点となる。
2. **置換え対象ページの決定**：主記憶に空きページがないため、OSはページ置換えアルゴリズム（LRUなど）に従って、追い出す（置き換える）対象ページを決定する。
3. **ページアウト**：置換え対象として決定されたページの内容が更新されている場合、そのページを補助記憶装置（ディスク）へ書き戻す。
4. **ページイン**：補助記憶装置から、アクセスしたかったページを主記憶上の空いた領域へ読み込む。

以上より、順序は「ページフォールト→置換え対象ページの決定→ページアウト→ページイン」となり、ウが正しい。

- ア・イは「置換え対象ページの決定」がページフォールトより前に行われており、順序が誤り（そもそもページフォールトが発生しないと置換えの必要性自体が判明しない）。
- エはページイン（読み込み）がページアウト（書き戻し）より先になっており、空きページがない状況では先に追い出し（ページアウト）をしなければ読み込む領域を確保できないため誤り。

**IPA公式：ウ**
