# 秋期 問7（基礎理論）

## 問題文

自然数をキーとするデータを，ハッシュ表を用いて管理する。キーxのハッシュ関数h（x）を

　　　　h（x）＝x mod n

とすると，任意のキーaとbが衝突する条件はどれか。ここで，nはハッシュ表の大きさであり，x mod nはxをnで割った余りを表す。

ア　a＋bがnの倍数

イ　a−bがnの倍数

ウ　nがa＋bの倍数

エ　nがa−bの倍数

## 使用画像

なし

## 解答と解説

**正解：イ**

ハッシュ関数 h(x) = x mod n は、xをnで割った余りを表す。キーaとbが衝突する（同じハッシュ値になる）条件は、

　h(a) = h(b)
　⇔ a mod n = b mod n
　⇔ (a − b) が nの倍数

である。これは合同式の性質から導かれる。例えばa=13, b=5, n=4のとき、13 mod 4 = 1, 5 mod 4 = 1で衝突するが、a−b=8はちょうどnの倍数（4×2）になっている。

- ア　a＋bがnの倍数：例えばa=1, b=3, n=4なら a+b=4でnの倍数だが、h(1)=1, h(3)=3で衝突しない。誤り。
- ウ・エ　「nがa＋bの倍数」「nがa−bの倍数」は、nの方が大きい値の約数になるという逆の関係であり、ハッシュ値が一致する条件とは無関係。

よって、衝突条件は「a−bがnの倍数」であるイが正解となる。

**IPA公式：イ**
