# 令和7年度秋期 問8（コンピュータシステム）

## 問題文

プロセッサの高速化技法の一つとして，同時に実行可能な複数の動作を，コンパイルの段階でまとめて一つの複合命令とし，高速化を図る方式はどれか。

ア　CISC

イ　MIMD

ウ　RISC

エ　VLIW

## 使用画像

なし

## 解答と解説

**正解：エ**

VLIW（Very Long Instruction Word）は、同時に実行可能な複数の命令（動作）を、実行時ではなくコンパイルの段階でまとめて一つの長い複合命令（超長命令語）として生成し、これを実行することでプロセッサの並列実行効率を高める高速化技法である。これは選択肢エの記述と一致する。

- アのCISC（Complex Instruction Set Computer）は複雑な命令セットを持つアーキテクチャの分類であり、複合命令をコンパイル時にまとめる方式ではない。
- イのMIMD（Multiple Instruction stream/Multiple Data stream）は、複数のプロセッサが異なる命令を異なるデータに対して並列実行する並列処理方式の分類である。
- ウのRISC（Reduced Instruction Set Computer）は命令セットを単純化・縮小したアーキテクチャの分類である。

**IPA公式：エ**
