# 令和7年度秋期 問40（技術要素）

## 問題文

ある組織ではCSIRTとPSIRTが設置されている。セキュリティインシデントのうち，PSIRTが対応するものはどれか。

ア　組織が顧客に販売した製品とその利用において発生したセキュリティインシデント

イ　組織が製品の在庫を管理するシステムにおいて発生したセキュリティインシデント

ウ　組織が製品の販売を管理するシステムにおいて発生したセキュリティインシデント

エ　組織の従業員の個人情報を管理するシステムにおいて発生したセキュリティインシデント

## 使用画像

なし

## 解答と解説

**正解：ア**

CSIRT（Computer Security Incident Response Team）は、組織内の情報システム（社内のITインフラや業務システムなど）で発生したセキュリティインシデントに対応するチームである。一方、PSIRT（Product Security Incident Response Team）は、組織が開発・販売する「製品」自体に関するセキュリティインシデント、すなわち製品の脆弱性や、製品が顧客の環境で使用された際に発生するセキュリティ上の問題に対応するチームである。

- アは、組織が顧客に販売した「製品」とその利用において発生したインシデントであり、これはまさにPSIRTが対応する対象である。
- イ・ウ・エは、いずれも組織内部の業務システム（在庫管理システム、販売管理システム、従業員の個人情報管理システム）で発生したインシデントであり、これらはCSIRTが対応する対象である。

**IPA公式：ア**
