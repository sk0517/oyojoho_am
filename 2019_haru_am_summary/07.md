# 平成31年度春期 問7（基礎理論）

## 問題文

JavaScriptの言語仕様のうち，オブジェクトの表記法などの一部の仕様を基にして規定したものであって，“名前と値の組みの集まり”と“値の順序付きリスト”の二つの構造に基づいてオブジェクトを表現する，データ記述の仕様はどれか。

ア　DOM

イ　JSON

ウ　SOAP

エ　XML

## 使用画像

なし

## 解答と解説

**正解：イ**

JSON（JavaScript Object Notation）は、JavaScriptのオブジェクトリテラル表記を基にしたデータ記述形式で、「名前と値の組みの集まり（オブジェクト、{key: value, ...}）」と「値の順序付きリスト（配列、[value, ...]）」という二つの構造でデータを表現する。軽量でテキストベースのため、Web APIなどでのデータ交換形式として広く使われている。

- ア：DOM（Document Object Model）はHTML/XML文書をツリー構造としてプログラムから操作するためのAPI仕様であり、データ記述形式ではない。
- ウ：SOAP（Simple Object Access Protocol）はXMLベースのメッセージ交換プロトコル。
- エ：XML（Extensible Markup Language）はタグによる階層構造でデータを記述するマークアップ言語であり、JavaScript由来の表記法ではない。

問題文の「名前と値の組みの集まり」＝オブジェクト、「値の順序付きリスト」＝配列という説明はJSONの構造そのものであるため、正解はイである。

**IPA公式：イ**
