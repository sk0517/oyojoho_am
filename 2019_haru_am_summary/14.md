# 平成31年度春期 問14（コンピュータシステム）

## 問題文

キャッシュサーバを利用した検索処理の平均応答時間は，キャッシュサーバでヒットした場合には0.2秒，ヒットしない場合には2.2秒である。現在の平均検索応答時間は，1.0秒である。3年後のキャッシュサーバのヒット率は，検索量の増加によって現状の半分になると予測されている。3年後の平均検索応答時間は何秒か。ここで，その他のオーバヘッドは考慮しない。

ア　1.1

イ　1.3

ウ　1.6

エ　1.9

## 使用画像

なし

## 解答と解説

**正解：ウ**

現在のヒット率をhとすると，平均応答時間の式は次のとおり。

0.2h + 2.2(1-h) = 1.0
2.2 - 2.0h = 1.0
h = 0.6

3年後はヒット率が現状の半分になるので，h' = 0.3。このときの平均応答時間は

0.2 × 0.3 + 2.2 × (1 - 0.3) = 0.06 + 1.54 = 1.6

よって3年後の平均検索応答時間は1.6秒となり，ウが正解である。

**IPA公式：ウ**
