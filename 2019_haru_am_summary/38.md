# 平成31年度春期 問38（技術要素）

## 問題文

パスワードクラック手法の一種である，レインボー攻撃に該当するものはどれか。

ア　何らかの方法で事前に利用者IDと平文のパスワードのリストを入手しておき，複数のシステム間で使い回されている利用者IDとパスワードの組みを狙って，ログインを試行する。

イ　パスワードに成り得る文字列の全てを用いて，総当たりでログインを試行する。

ウ　平文のパスワードとハッシュ値をチェーンによって管理するテーブルを準備しておき，それを用いて，不正に入手したハッシュ値からパスワードを解読する。

エ　利用者の誕生日や電話番号などの個人情報を言葉巧みに聞き出して，パスワードを類推する。

## 使用画像

なし

## 解答と解説

**正解：ウ**

レインボー攻撃（レインボーテーブル攻撃）は、あらかじめ大量の平文パスワードとそれに対応するハッシュ値の組を、計算量を削減する連鎖（チェーン）構造で効率的にまとめた「レインボーテーブル」を準備しておき、不正に入手したハッシュ値からテーブルを逆引きして元のパスワードを高速に解読する手法である。総当たり攻撃に比べて必要な計算・記憶コストを抑えられるのが特徴。

- ア：パスワードリスト攻撃（使い回しパスワードを狙う攻撃）の説明。
- イ：ブルートフォース攻撃（総当たり攻撃）の説明。
- エ：ソーシャルエンジニアリングの説明。

**IPA公式：ウ**
