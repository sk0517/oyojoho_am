# 平成30年度春期 問11（コンピュータシステム）

## 問題文

メモリインタリーブの説明として，適切なものはどれか。

ア　主記憶と外部記憶を一元的にアドレス付けし，主記憶の物理容量を超えるメモリ空間を提供する。

イ　主記憶と磁気ディスク装置との間にバッファメモリを置いて，双方のアクセス速度の差を補う。

ウ　主記憶と入出力装置との間でCPUとは独立にデータ転送を行う。

エ　主記憶を複数のバンクに分けて，CPUからのアクセス要求を並列的に処理できるようにする。

## 使用画像

なし

## 解答と解説

**正解：エ**

メモリインタリーブとは、主記憶（メインメモリ）を複数の独立したバンク（モジュール）に分割し、CPUからの連続したアドレスへのアクセス要求を複数バンクに振り分けて並列にアクセスすることで、実効的なメモリアクセス速度を向上させる技術である。1つのバンクがアクセス中でも別のバンクへのアクセスを同時に開始できるため、メモリアクセスのスループットが向上する。これはエの「主記憶を複数のバンクに分けて，CPUからのアクセス要求を並列的に処理できるようにする」という記述に合致する。

ア は仮想記憶（バーチャルメモリ）の説明であり、主記憶と補助記憶を組み合わせて物理容量を超えるアドレス空間を提供する仕組みである。

イ はディスクキャッシュの説明で、主記憶と磁気ディスクの速度差を緩衝するバッファメモリの働きを指す。

ウ はDMA（Direct Memory Access）の説明で、CPUを介さずに入出力装置と主記憶間で直接データ転送を行う仕組みである。

以上より、メモリインタリーブの説明として適切なのはエである。

**IPA公式：エ**
