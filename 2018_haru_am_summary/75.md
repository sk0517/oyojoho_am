# 平成30年度春期 問75（ストラテジ）

## 問題文

ビッグデータ分析の手法の一つであるデシジョンツリーを活用してマーケティング施策の判断に必要な事象を整理し，発生確率の精度を向上させた上で二つのマーケティング施策a，bの選択を行う。マーケティング施策を実行した場合の利益増加額（売上増加額−費用）の期待値が最大となる施策と，そのときの利益増加額の期待値の組合せはどれか。

## 解答と解説

**正解：ウ**

デシジョンツリーでは、意思決定の分岐点（□）では期待値が最大となる枝を選択し、発生確率の分岐点（○）では確率で加重平均した期待値を計算する。末端から順にさかのぼって計算する。

**施策a（初期費用30億円）**
- 追加費用60億円が発生した場合の期待値：0.3×200 + 0.7×100 = 60 + 70 = 130億円。追加費用60億円を差し引くと 130 − 60 = 70億円。
- 追加費用なしの場合：50億円。
- 意思決定点では有利な方（70億円）を選択する。
- 発生確率0.4／0.6で加重：0.4×70 + 0.6×120 = 28 + 72 = 100億円。
- 初期費用30億円を差し引き、施策aの期待利益増加額 = 100 − 30 = 70億円。

**施策b（初期費用40億円）**
- 追加費用40億円が発生した場合の期待値：0.4×150 + 0.6×100 = 60 + 60 = 120億円。追加費用40億円を差し引くと 120 − 40 = 80億円。
- 追加費用なしの場合：70億円。
- 意思決定点では有利な方（80億円）を選択する。
- 発生確率0.3／0.7で加重：0.3×80 + 0.7×140 = 24 + 98 = 122億円。
- 初期費用40億円を差し引き、施策bの期待利益増加額 = 122 − 40 = 82億円。

施策aの期待値70億円と施策bの期待値82億円を比較すると、施策bの方が大きいため、施策bを選択し、期待値は82億円となる。これは表中のウ「b、82」に一致する。

**IPA公式：ウ**
