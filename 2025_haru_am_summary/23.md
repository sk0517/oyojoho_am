# 令和7年度春期 問23（技術要素）

## 問題文

W3Cで仕様が定義され，矩（く）形や円，直線，文字列などの図形オブジェクトをXML形式で記述し，Webページでの図形描画にも使うことができる画像フォーマットはどれか。

ア　OpenGL

イ　PNG

ウ　SVG

エ　TIFF

## 使用画像

なし

## 解答と解説

**正解：ウ**

SVG（Scalable Vector Graphics）は、W3Cが仕様を策定したベクター画像フォーマットで、矩形・円・直線・文字列などの図形オブジェクトをXML形式のテキストで記述する。拡大・縮小しても画質が劣化しないベクター形式であり、HTML5と親和性が高くWebページ上での図形描画にも直接利用できる。

他の選択肢は誤り。アのOpenGLは3Dグラフィックス描画のためのAPI（プログラミングインタフェース）であり、画像フォーマットではない。イのPNG（Portable Network Graphics）とエのTIFF（Tagged Image File Format）はいずれもビットマップ（ラスター）形式の画像フォーマットであり、XMLで図形を記述するものではない。

**IPA公式：ウ**
