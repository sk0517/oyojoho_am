# 令和7年度春期 問7（基礎理論）

## 問題文

fact（n）は，非負の整数nに対してnの階乗を返す。fact（n）の再帰的な定義はどれか。

ア　if　n＝0　then　return　0　else　return　n×fact（n−1）

イ　if　n＝0　then　return　0　else　return　n×fact（n＋1）

ウ　if　n＝0　then　return　1　else　return　n×fact（n−1）

エ　if　n＝0　then　return　1　else　return　n×fact（n＋1）

## 使用画像

なし

## 解答と解説

**正解：ウ**

階乗の定義は，0の階乗が1（0!＝1）であり，n≧1のときはn!＝n×(n−1)!である。再帰的定義ではこの2つの条件（基底部と再帰部）を満たす必要がある。

基底部：n＝0のとき，fact(0)は1を返す必要がある（0!＝1のため）。
再帰部：n≠0のとき，fact(n)＝n×fact(n−1)を返す必要がある（nを1つずつ減らして0に近づけていくことで再帰が終了する）。

選択肢ウ「if n＝0 then return 1 else return n×fact(n−1)」はこの両方を満たしている。

ア・イ）n＝0のとき0を返しており，0!＝1という定義に反する（この時点で誤り）。
エ）fact(n＋1)のようにnを増加させると，再帰が0に到達せず無限に続いてしまう（停止しない）。

**IPA公式：ウ**
