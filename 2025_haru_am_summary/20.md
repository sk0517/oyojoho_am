# 令和7年度春期 問20（コンピュータシステム）

## 問題文

FPGAなどに実装するデジタル回路を記述し，論理合成するために使用される言語はどれか。

ア　DDL

イ　HDL

ウ　UML

エ　XML

## 使用画像

なし

## 解答と解説

**正解：イ**

HDL（Hardware Description Language，ハードウェア記述言語）は，デジタル回路の構造や動作をテキストで記述するための言語であり，VerilogHDLやVHDLが代表例である。FPGAやASICの設計では，HDLで記述した回路をEDAツールで論理合成し，実際のゲート回路（ネットリスト）へと変換する。よってイが正しい。

ア）DDL（Data Definition Language）はデータベースのテーブルなどを定義するSQL言語の一種であり，回路記述とは無関係。誤り。
ウ）UML（Unified Modeling Language）はソフトウェアの設計を表現するモデリング言語であり，回路の論理合成には使用されない。誤り。
エ）XML（Extensible Markup Language）はデータを構造化して記述するマークアップ言語であり，回路記述言語ではない。誤り。

**IPA公式：イ**
