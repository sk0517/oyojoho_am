# 令和7年度春期 問9（コンピュータシステム）

## 問題文

DMAコントローラーの説明として，適切なものはどれか。

ア　MPUでは時間が掛かる積和演算を，高速に行う。

イ　仮想メモリ機能，メモリ保護機能などのメモリ管理機能を提供する。

ウ　動作クロックに合わせてカウントするカウントレジスタをもち，それによって時間の経過を保持する。

エ　メモリと入出力装置，又はメモリとメモリとの間のデータ転送を，MPUを介さずに行う。

## 使用画像

なし

## 解答と解説

**正解：エ**

DMA（Direct Memory Access）コントローラーは，メモリと入出力装置間，またはメモリ間のデータ転送を，CPU（MPU）を介さずに直接行う装置である。これにより，大量のデータ転送中もCPUは他の処理を並行して実行でき，システム全体の処理効率が向上する。よってエが正しい。

ア）積和演算を高速に行うのはDSP（デジタル信号処理プロセッサ）やFPU，GPUなどの説明であり，DMAコントローラーの機能ではない。誤り。
イ）仮想メモリ機能やメモリ保護機能を提供するのはMMU（メモリ管理ユニット）の説明である。誤り。
ウ）動作クロックに合わせて時間経過をカウントするのはタイマ（インターバルタイマ）の説明である。誤り。

**IPA公式：エ**
