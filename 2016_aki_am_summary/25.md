# 平成28年度秋期 問25（技術要素）

## 問題文

動画や音声などのマルチメディアコンテンツのレイアウトや再生のタイミングをXMLフォーマットで記述するためのW3C勧告はどれか。

ア　Ajax

イ　CSS

ウ　SMIL

エ　SVG

## 使用画像

なし

## 解答と解説

**正解：ウ**

SMIL（Synchronized Multimedia Integration Language：シミル）は、動画・音声・画像・テキストなどの複数のマルチメディアコンテンツについて、画面上のレイアウト（配置）や再生のタイミング（同期）をXML形式で記述するためのマークアップ言語であり、W3Cによって勧告されている。

各選択肢を確認する。

ア「Ajax」は誤り。Ajax（Asynchronous JavaScript and XML）は、Webブラウザ上でページ全体を再読み込みせずにサーバーと非同期通信を行い、画面の一部を動的に更新するための技術・手法であり、マルチメディアの再生タイミングを記述するものではない。

イ「CSS」は誤り。CSS（Cascading Style Sheets）は、HTML文書などの見た目（レイアウト、色、フォントなど）を指定するためのスタイルシート言語であり、マルチメディアの時間的な同期制御を目的としたものではない。

ウ「SMIL」は正しい。前述のとおり、マルチメディアコンテンツの空間的配置（レイアウト）と時間的な再生タイミングをXMLで記述するためのW3C勧告であり、設問の内容と合致する。

エ「SVG」は誤り。SVG（Scalable Vector Graphics）は、ベクター形式の2次元グラフィックスをXMLで記述するためのフォーマットであり、静止画・図形の表現が主目的である。

以上より、正解はウである。

**IPA公式：ウ**
