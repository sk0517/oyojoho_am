# 令和2年度秋期 問21（コンピュータシステム）

## 問題文

FPGAなどに実装するディジタル回路を記述して，直接論理合成するために使用されるものはどれか。

ア　DDL

イ　HDL

ウ　UML

エ　XML

## 使用画像

なし

## 解答と解説

**正解：イ**

FPGA（Field Programmable Gate Array）などのディジタル回路を記述し，そのまま論理合成（回路の構造へ変換）する目的で使われる言語がHDL（Hardware Description Language：ハードウェア記述言語）である。代表的なHDLにはVHDLやVerilog HDLがあり，回路の動作や構造を記述してシミュレーションや論理合成に用いる。

- ア　DDL（Data Definition Language）はデータベースのスキーマ定義言語であり，回路記述とは無関係。
- ウ　UMLはソフトウェアやシステムの設計を図示するモデリング言語であり，論理合成には使用しない。
- エ　XMLはデータ表現用のマークアップ言語であり，回路の論理合成には用いない。

**IPA公式：イ**
