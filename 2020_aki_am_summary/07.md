# 令和2年度秋期 問7（基礎理論）

## 問題文

オブジェクト指向のプログラム言語であり，クラスや関数，条件文などのコードブロックの範囲はインデントの深さによって指定する仕様であるものはどれか。

ア　JavaScript

イ　Perl

ウ　Python

エ　Ruby

## 使用画像

なし

## 解答と解説

**正解：ウ**

Pythonはオブジェクト指向言語であり、他の多くの言語がブロックの範囲を波括弧`{}`やend文などで明示するのに対し、Pythonはインデント（字下げ）の深さでクラス・関数・条件文などのコードブロックの範囲を決定するという特徴的な文法仕様を持つ。

- ア：JavaScriptは波括弧`{}`でブロックを表す。
- イ：Perlも波括弧`{}`でブロックを表す。
- エ：Rubyはオブジェクト指向言語だが、ブロックの範囲は`end`キーワードで明示し、インデントは構文上必須ではない。

**IPA公式：ウ**
