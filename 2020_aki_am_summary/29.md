# 令和2年度秋期 問29（技術要素）

## 問題文

“東京在庫”表と“大阪在庫”表に対して，SQL文を実行して得られる結果はどれか。ここで，実線の下線は主キーを表す。

〔SQL文〕

SELECT 商品コード, 在庫数 FROM 東京在庫

　　　　UNION ALL

SELECT 商品コード, 在庫数 FROM 大阪在庫
