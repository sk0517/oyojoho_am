# 令和3年度春期 問27（技術要素）

## 問題文

W3Cで仕様が定義され，矩（く）形や円，直線，文字列などの図形オブジェクトをXML形式で記述し，Webページでの図形描画にも使うことができる画像フォーマットはどれか。

ア　OpenGL

イ　PNG

ウ　SVG

エ　TIFF

## 使用画像

なし

## 解答と解説

**正解：ウ**

SVG（Scalable Vector Graphics）は、W3Cによって仕様が策定されたXMLベースのベクター画像フォーマットである。矩形、円、直線、文字列などの図形オブジェクトをXMLのタグとして記述し、拡大・縮小しても画質が劣化しないベクター描画をWebページ上で実現できる。

- ア：OpenGLは3Dグラフィックス描画用のAPI（プログラミングインタフェース）であり、画像フォーマットではない。
- イ：PNG（Portable Network Graphics）はラスター（ビットマップ）形式の画像フォーマットであり、XMLで図形を記述するものではない。
- エ：TIFF（Tagged Image File Format）もラスター形式の画像フォーマットであり、印刷・スキャン分野で使われるが、XML記述のベクター形式ではない。

したがって、XML形式で図形オブジェクトを記述するウが適切である。

**IPA公式：ウ**
