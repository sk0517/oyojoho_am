# 令和3年度春期 問20（コンピュータシステム）

## 問題文

Hadoopの説明はどれか。

ア　Java EE仕様に準拠したアプリケーションサーバ

イ　LinuxやWindowsなどの様々なプラットフォーム上で動作するWebサーバ

ウ　機能の豊富さが特徴のRDBMS

エ　大規模なデータセットを分散処理するためのソフトウェアライブラリ

## 使用画像

なし

## 解答と解説

**正解：エ**

Hadoopは，大規模なデータセットを多数のサーバ（コモディティサーバのクラスタ）に分散させ，分散処理によって効率的に処理するためのオープンソースのソフトウェアライブラリ（フレームワーク）である。分散ファイルシステムHDFSと，MapReduceに代表される分散処理の仕組みを中核とし，ビッグデータ処理の基盤として広く利用されている。

アはJava EEに準拠したアプリケーションサーバ（例：GlassFishなど）の説明，イはクロスプラットフォームで動作するWebサーバ（例：Apache HTTP Serverなど）の説明，ウは機能の豊富なRDBMS（例：Oracle Databaseなど）の説明であり，いずれもHadoopの説明としては誤りである。

**IPA公式：エ**
