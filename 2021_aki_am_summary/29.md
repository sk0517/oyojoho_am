# 令和3年度秋期 問29（技術要素）

## 問題文

“部門別売上”表から，部門コードごと，期ごとの売上を得るSQL文はどれか。

ア　SELECT 部門コード, '第1期' AS 期, 第1期売上 AS 売上

　　  FROM 部門別売上

　　  INTERSECT

　　  (SELECT 部門コード, '第2期' AS 期, 第2期売上 AS 売上

　　    FROM 部門別売上)

　　  ORDER BY 部門コード, 期

イ　SELECT 部門コード, '第1期' AS 期, 第1期売上 AS 売上

　　  FROM 部門別売上

　　  UNION

　　  (SELECT 部門コード, '第2期' AS 期, 第2期売上 AS 売上

　　    FROM 部門別売上)

　　  ORDER BY 部門コード, 期

ウ　SELECT A.部門コード, '第1期' AS 期, A.第1期売上 AS 売上

　　  FROM 部門別売上 A

　　  CROSS JOIN

　　  (SELECT B.部門コード, '第2期' AS 期, B.第2期売上 AS 売上

　　       FROM 部門別売上 B) T

　　  ORDER BY 部門コード, 期

エ　SELECT A.部門コード, '第1期' AS 期, A.第1期売上 AS 売上

　　  FROM 部門別売上 A

　　  INNER JOIN

　　  (SELECT B.部門コード, '第2期' AS 期, B.第2期売上 AS 売上

　　       FROM 部門別売上 B) T ON A.部門コード = T.部門コード

　　  ORDER BY 部門コード, 期

## 解答と解説

**正解：イ**

画像の〔問合せ結果〕は，部門コードごとに「第1期」と「第2期」の行が縦に並び，部門コード・期・売上の3列から成る形式である。これは，第1期のデータを（部門コード, 期='第1期', 売上=第1期売上）の3列に整形したSELECT結果と，第2期のデータを同様に整形したSELECT結果とを，行方向に結合することで得られる。

この「異なるSELECT結果を行方向に積み重ねる」操作を行うのがUNION（集合演算子）である。UNIONは両方のSELECT結果を合成し重複行を除去して1つの結果集合とするため，部門コードごとに第1期・第2期の売上行が並ぶ問合せ結果と一致する。

アのINTERSECTは共通行のみを抽出する演算であり，期の列の値が異なるため共通行は存在せず結果が空になってしまう。ウのCROSS JOINとエのINNER JOINは行同士を列方向に結合（横に連結）する演算であり，1行あたりの列数が増えるだけで，期ごとに行を分けた画像の結果構造にはならない。
