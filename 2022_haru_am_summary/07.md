# 令和4年度春期 問7（基礎理論）

## 問題文

プログラム言語のうち，ブロックの範囲を指定する方法として特定の記号や予約語を用いず，等しい文字数の字下げを用いるという特徴をもつものはどれか。

ア　C

イ　Java

ウ　PHP

エ　Python

## 使用画像

なし

## 解答と解説

**正解：エ**

ブロック（コードのまとまり）の範囲を，波括弧｛｝やbegin～end，あるいはend文などの特定の記号・予約語を使わずに，字下げ（インデント）の深さによって表現するプログラミング言語はPythonである。Pythonでは同じブロックに属する文は同じ文字数のインデントで揃える必要があり，インデントの深さがそのままブロック構造（if文やfor文の範囲など）を表す文法上の意味を持つ。

ア「C」はブロックを｛｝で明示的に区切る。
イ「Java」も｛｝でブロックを区切る言語である。
ウ「PHP」も｛｝（および一部endif等の代替構文）でブロックを区切る。

これらに対し，エ「Python」のみがインデントをブロックの構文的な区切りとして採用しており，インデントを誤るとエラーになる特徴的な言語である。

したがって正解はエである。

**IPA公式：エ**
