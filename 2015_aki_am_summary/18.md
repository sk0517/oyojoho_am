# 平成27年度秋期 問18（コンピュータシステム）

## 問題文

三つの媒体A〜Cに次の条件でファイル領域を割り当てた場合，割り当てた領域の総量が大きい順に媒体を並べたものはどれか。

〔条件〕

（1）ファイル領域を割り当てる際の媒体選択アルゴリズムとして，空き領域が最大の媒体を選択する方式を採用する。

（2）割当て要求されるファイル領域の大きさは，順に90，30，40，40，70，30（Mバイト）であり，割り当てられたファイル領域は，途中で解放されない。

（3）各媒体は容量が同一であり，割当て要求に対して十分な大きさをもち，初めは全て空きの状態である。

（4）空き領域の大きさが等しい場合には，A，B，Cの順に選択する。

ア　A，B，C

イ　A，C，B

ウ　B，A，C

エ　C，B，A

## 使用画像

なし

## 解答と解説

**正解：エ**

媒体選択アルゴリズムは「空き領域が最大の媒体を選択する」方式であり、空き領域が等しい場合はA、B、Cの順に選択する。各媒体の容量は同一（Sとする）で初期状態は全て空きとする。要求は順に90、30、40、40、70、30（Mバイト）である。

初期状態: A=S, B=S, C=S（全て等しいのでAが選ばれる）

1. 90要求 → 空き最大はA,B,C同値→Aを選択。A=S-90
2. 30要求 → 空き最大はB,C（S）→Bを選択。B=S-30
3. 40要求 → 空き最大はC（S）→Cを選択。C=S-40
4. 40要求 → 空き比較：A=S-90, B=S-30, C=S-40 → 最大はB(S-30)→Bを選択。B=S-70
5. 70要求 → 空き比較：A=S-90, B=S-70, C=S-40 → 最大はC(S-40)→Cを選択。C=S-110
6. 30要求 → 空き比較：A=S-90, B=S-70, C=S-110 → 最大はB(S-70)→Bを選択。B=S-100

最終的な割当て済み領域（使用量）の合計:
- A = 90
- B = 30+40+30 = 100
- C = 40+70 = 110

割り当てた領域の総量が大きい順に並べると、C(110) ＞ B(100) ＞ A(90) となる。

- ア〜ウ　いずれも上記計算結果と一致しない。
- エ　正しい。C，B，Aの順である。

以上より、正解はエである。

**IPA公式：エ**
