# 平成27年度秋期 問29（技術要素）

## 問題文

“倉庫別商品在庫集計”表から在庫数の合計を求めたい。倉庫番号'C003'の倉庫で在庫数が100以上の商品に対して，全ての倉庫における在庫数の合計を求めるSQL文のaに入る適切な字句はどれか。ここで，該当する商品は複数存在するとともに在庫数が100未満の商品も存在するものとする。また，実線の下線は主キーを表す。

倉庫別商品在庫集計（倉庫番号，商品コード，在庫数）

〔SQL文〕

　　　　SELECT 商品コード, SUM(在庫数) AS 在庫合計 FROM 倉庫別商品在庫集計

　　　　　　　　WHERE    a   

　　　　　　　　GROUP BY 商品コード

ア　商品コード = (SELECT 商品コード FROM 倉庫別商品在庫集計

　　　　WHERE 倉庫番号 = 'C003' AND 在庫数 >= 100)

イ　商品コード = ALL (SELECT 商品コード FROM 倉庫別商品在庫集計

　　　　WHERE 倉庫番号 = 'C003' AND 在庫数 >= 100)

ウ　商品コード　IN (SELECT 商品コード FROM 倉庫別商品在庫集計

　　　　WHERE 倉庫番号 = 'C003' AND 在庫数 >= 100)

エ　EXISTS (SELECT * FROM 倉庫別商品在庫集計

　　　　WHERE 倉庫番号 = 'C003' AND 在庫数 >= 100)

## 使用画像

なし

## 解答と解説

**正解：ウ**

求めたいのは「倉庫番号'C003'で在庫数が100以上である商品コード」に該当する商品について、その商品コードごとに全倉庫分の在庫数を合計した値である。つまり、外側の集約（SUM・GROUP BY）は全倉庫の行を対象にしつつ、対象とする商品コードの絞り込みだけをサブクエリで行う必要がある。

これを実現するには、副問合せで対象となる商品コードの集合を求め、外側のWHERE句でその集合に含まれる商品コードの行（全倉庫分）を抽出すればよい。「商品コード IN (副問合せ)」は、副問合せが返す商品コードの集合のいずれかに一致する行を全て（倉庫を問わず）選択するため、要求どおりの絞り込みができる。

他の選択肢を確認する。

- ア　「＝ (副問合せ)」は、副問合せの結果が単一の値であることを要求するスカラ副問合せの書き方であり、該当する商品が複数存在する（問題文の前提）場合はエラーとなり、要件を満たさない。
- イ　「＝ ALL (副問合せ)」は、外側の値が副問合せの返す全ての値と等しいことを要求する演算子であり、複数の異なる商品コードが返る場合は意図した絞り込みにならず、通常は該当行がなくなってしまう。要件を満たさない。
- エ　「EXISTS (副問合せ)」は、相関副問合せとして使う場合に有効であるが、ここでは外側のクエリとの相関条件（例：外側.商品コード＝内側.商品コード）が記述されておらず、単独では対象商品コードの絞り込みとして機能しない。要件を満たさない。

以上より、正しい絞り込みができるのはウの「商品コード IN (副問合せ)」である。

**IPA公式：ウ**
