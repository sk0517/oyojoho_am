# 平成27年度秋期 問15（コンピュータシステム）

## 問題文

MTBFがx時間，MTTRがy時間のシステムがある。使用条件が変わったので，MTBF，MTTRがともに従来の1.5倍になった。新しい使用条件での稼働率はどうなるか。

ア　x，yの値によって変化するが，従来の稼働率よりは大きい値になる。

イ　従来の稼働率と同じ値である。

ウ　従来の稼働率の1.5倍になる。

エ　従来の稼働率の2／3倍になる。

## 使用画像

なし

## 解答と解説

**正解：イ**

稼働率は次式で求められる。

稼働率 = MTBF ÷ (MTBF + MTTR)

従来の稼働率は x ÷ (x + y) である。

MTBF、MTTRがともに1.5倍になった場合の新しい稼働率は、

新MTBF = 1.5x、新MTTR = 1.5y

新しい稼働率 = 1.5x ÷ (1.5x + 1.5y) = 1.5x ÷ 1.5(x + y) = x ÷ (x + y)

これは従来の稼働率 x ÷ (x + y) と完全に一致する。すなわち、MTBFとMTTRが同じ比率で変化する場合、分子・分母がともに同じ倍率でスケールするため、比率としての稼働率は変化しない。

- ア　誤り。x，yの値に関わらず比率は変わらないため、大きい値になるとは限らない。
- イ　正しい。従来の稼働率と同じ値になる。
- ウ　誤り。1.5倍になるのはMTBFとMTTRそれぞれの値であり、稼働率という比率自体は変化しない。
- エ　誤り。同様に2／3倍になることもない。

以上より、正解はイである。

**IPA公式：イ**
