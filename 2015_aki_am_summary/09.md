# 平成27年度秋期 問9（コンピュータシステム）

## 問題文

複数のデータに対して1個の命令で同一の操作を同時並列に行う方式で，マルチメディアデータなどを扱うCPUに採用されているものはどれか。

ア　MIMD

イ　MISD

ウ　SIMD

エ　SISD

## 使用画像

なし

## 解答と解説

**正解：ウ**

フリンの分類（Flynn's taxonomy）における並列処理方式の名称を問う問題である。

- SISD（Single Instruction Single Data）：単一命令で単一データを処理する、従来型の逐次処理方式。
- SIMD（Single Instruction Multiple Data）：単一の命令で複数のデータに対して同時に同一の演算を行う方式。画像・音声処理などマルチメディアデータのように、同種の演算を大量のデータへ一斉に適用する処理に適しており、多くのCPUのマルチメディア拡張命令（SSEやAVXなど）に採用されている。
- MISD（Multiple Instruction Single Data）：複数の命令で単一のデータを処理する方式で、実用例は少ない。
- MIMD（Multiple Instruction Multiple Data）：複数の命令で複数のデータを独立に処理する方式で、マルチコアCPUや分散システムなどに相当する。

設問の「複数のデータに対して1個の命令で同一の操作を同時並列に行う方式」はSIMDの定義そのものであり、マルチメディアデータを扱うCPUに採用されているという記述とも合致する。したがってウが正解である。

**IPA公式：ウ**
