# 平成27年度秋期 問6（基礎理論）

## 問題文

次に示すユークリッドの互除法（方法1，方法2）で，正の整数a，bの最大公約数は，それぞれmとnのどちらの変数に求まるか。ここで，m mod nは，mをnで割った余りを表す。

## 解答と解説

**正解：ウ**

ユークリッドの互除法は、m mod n を計算し、その余りが0になるまで「n→m、r→n」と置き換えながら m mod n → r を繰り返すアルゴリズムであり、余りが0になった時点の除数（一つ前のn、置き換え後のm）が最大公約数となる。

具体例（a＝12, b＝8）で両方法を追跡して確認する。

**方法1**：a→m(12), b→n(8) → m mod n → r (r＝4) → ループ1判定(r＝0か？No) → n→m(m＝8) → r→n(n＝4) → m mod n→r (8 mod 4＝0, r＝0) → ループ1判定(r＝0か？Yes、終了)。このときm＝8、n＝4であり、m mod n を計算する前にn→m、r→nの代入が行われてからループ判定に戻る構造なので、最終的に最大公約数（4）はmに格納されている。

**方法2**：a→m(12), b→n(8) → ループ2判定(初回はrが未定義のためまず内部処理へ) → m mod n→r (r＝4) → n→m(m＝8) → r→n(n＝4) → ループ2判定(r＝0か？No、繰り返す) → m mod n→r (8 mod 4＝0) → n→m(m＝4) → r→n(n＝0) → ループ2判定(r＝0か？Yes、終了)。このときn＝4であり、最大公約数はnに格納されている。

したがって、方法1ではm、方法2ではnに最大公約数が求まり、ウが正解である。

**IPA公式：ウ**
