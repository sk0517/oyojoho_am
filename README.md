# 応用情報技術者試験 午前問題集（2015〜2025）

応用情報技術者試験（AP）の**午前問題 全21期・1,680問**を、問題文・選択肢・図表・
**解答と解説**付きの Markdown にまとめ、ブラウザで学習できるビューアを備えたリポジトリです。

- 収録範囲: **2015年（平成27年）春期 〜 2025年（令和7年）秋期**（春・秋 各80問、計21期）
  ※ 2020年春期は新型コロナで中止のため無し
- 数式・記号・下線・分数・ソースコード・図表は、元データ（`origin/`）の HTML を正として忠実に再現
- 解答の正解記号は **IPA 公式解答（`answer/` の PDF）** を唯一の根拠として機械照合済み

> ⚠️ 解説文は学習用に独自作成したものです。正解記号は IPA 公式解答に基づきますが、
> 解説の内容は公式のものではありません。誤りに気づいたら Issue や修正で直してください。

---

## 📱 ビューアで学習する

`viewer/index.html` がクイズビューアです。年度・季節を選び、問題を解いて
「解答を見る」ボタンで解答・解説を表示できます。

### 使い方（主な機能）
- **年度・季節の選択** … プルダウンで切り替え（前回の続きから自動再開）
- **📋 一覧** … その期の全問をリスト表示、タップで学習表示へジャンプ
- **カテゴリ / タグで絞り込み** … 「最優先」「高頻出」などの優先度タグでも絞り込み可
- **🔀 シャッフル** … 出題順をランダム化
- **既読記録** … ブラウザの localStorage に保存（端末ごと）
- 数式は **KaTeX** で表示（分数・指数・上線など）

### 開き方
| 方法 | 手順 |
|---|---|
| PC でローカル | `viewer/index.html` をブラウザで開くだけ |
| スマホ / どこからでも | GitHub Pages で公開 → `https://sk0517.github.io/oyojoho_am/viewer/index.html` |
| 同一Wi-Fi | プロジェクト直下で `python -m http.server 8123` → スマホから `http://<PCのIP>:8123/viewer/index.html` |

## 📝 問題ファイル（Markdown）の書式

各 `NN.md` はフロントマター＋3セクションで構成されます。

```markdown
---
年度: 令和7年度
時期: 春期
区分: 午前
問題番号: 問1
カテゴリ: テクノロジ系      # テクノロジ系 / マネジメント系 / ストラテジ系
難易度: ★★★
重要度: ★★
タグ: 基礎理論, 論理回路, 最優先   # 分野タグ＋優先度タグ
---

# 令和7年度春期 問1（基礎理論）

## 問題文
（問題文・選択肢ア〜エ。数式は $…$、下線は <u>…</u> 等で表現）

## 使用画像
![…](images/….gif)

## 解答と解説
**正解：エ**
（解説）
**IPA公式：エ**
```

### 記号・書式の表現（ビューアの KaTeX / HTML で描画）
| 元データ | Markdown 表現 | 例 |
|---|---|---|
| 数式・分数・指数・下付き | `$…$`（TeX） | `$\frac{M!}{(M-n)!}$`, `$2^{n-1}$`, `$N_A$` |
| 否定・補集合の上線 | `\overline{}` | `$\overline{A}\cdot\overline{B}$` |
| 実線の下線（主キー等） | `<u>…</u>` | `<u>商品コード</u>` |
| 破線の下線（外部キー） | `<u class="dashed">…</u>` | `<u class="dashed">メーカコード</u>` |
| 添字ラベル（下線部 a〜d） | `<sub>…</sub>` | `<sub>a</sub><u>再帰的</u>` |
| 空欄の枠 | `[　a　]` | `WHERE [　a　]` |

### 優先度タグ（`出題傾向メモ.md` に基づく）
`最優先`（皆勤・高頻出トピック）/ `高頻出` / `増加傾向` / `減少傾向` /
`頻出(2回)` `頻出(3回)` `頻出(4回以上)`（再出題回数）

## 🔧 データを再生成・検証する

```bash
# 問題データ (data.js) を再生成（MD を編集したあと）
python viewer/build_viewer.py

# origin と MD を照合して書式崩れを監査
python verify_origin.py

# origin を正として問題文を TeX 記法に再変換（対象を指定）
python fix_tex.py 2025_aki/01 2015_haru/38 --apply
```

正解記号は `answer/answers.json`（`answer/extract_answers.py` で公式PDFから抽出）を正として、
全問で照合済みです。
